Overview

Designed and brought up MCU-based sensor test and data-collection platforms to evaluate next-generation MEMS inertial sensors at sampling rates exceeding 12.8 kHz. The platform was developed for internal characterization and validation workflows and emphasized flexibility, throughput, and measurement integrity.

Due to the proprietary nature of this work, implementation details and hardware artifacts cannot be publicly disclosed.

Design Constraints

  • Support multiple prototype sensors with evolving interfaces
  • Sustain high-throughput data capture without loss
  • Enable both real-time streaming and onboard logging
  • Provide robust timing and synchronization for sensor evaluation

System Architecture

The platform centered around an MCU-based architecture with onboard memory and DMA-driven data acquisition pipelines. Multiple communication interfaces were implemented to support flexible integration with laboratory equipment and automated test setups.

  • MCU-based control and acquisition firmware written in C
  • Onboard memory and SD card storage for high-rate logging
  • DMA-based data movement to minimize CPU overhead
  • Interfaces including FDCAN, SPI, I²C, UART, and USB3300 high-speed USB

Firmware and Data Pipeline

Firmware was designed to support concurrent acquisition from multiple sensors while maintaining deterministic timing. Interrupt-driven control paths were combined with DMA transfers to achieve reliable high-rate sampling.

  • Configurable sensor drivers for rapid evaluation of new devices
  • Real-time streaming to host systems for live monitoring
  • Buffered logging for long-duration characterization runs

Data Analysis and Characterization

Captured sensor data was analyzed offline using Python and MATLAB to extract noise and performance metrics relevant to inertial navigation and compassing applications.

  • Allan deviation analysis for bias instability and noise density
  • Time- and frequency-domain inspection of sensor outputs
  • Cross-sensor comparison under identical excitation conditions

Outcomes

  • Validated performance of multiple next-generation MEMS IMUs
  • Enabled repeatable, high-rate sensor characterization workflows
  • Contributed measurement methodology and results to internal research and publications
