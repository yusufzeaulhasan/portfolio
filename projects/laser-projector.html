Overview

Collaborated on a 2-DoF motor-controlled laser projection system capable of rendering still images and animations. The system combined embedded firmware on an ESP32 with digital logic in SystemVerilog to support closed-loop control and timing-accurate coordination.

Challenges

  • Achieve stable closed-loop motor control
  • Coordinate real-time timing between MCU and logic
  • Produce repeatable projection trajectories

System

  • ESP32 firmware with ISR-based scheduling
  • SPI link to digital logic decoder
  • 2-axis actuation + trajectory generation

Outcome

  • Projected multiple animations in real time
  • Implemented PID control loop tuning
  • Validated end-to-end system integration
